# 交通系ICカード管理システム：ピッすい かんたん導入ガイド

**バージョン**: 2.7.0
**最終更新日**: 2026年4月

---

**4月から正式に使い始める場合、面倒な作業は一切不要です。3月中に準備しておきましょう。**

---

## おすすめスケジュール

```
【3月中】                          【4月1日～】

 インストール（数分）                 そのまま普段どおり使うだけ！
    ↓                               ・利用履歴が自動で記録されます
 職員・カードの登録                   ・物品出納簿もボタン1つで出力
    ↓
 試しに貸出・返却してみる
```

3月中の操作は練習として記録されます。4月になると自動的に「前年度繰越」として処理されるため、特別な操作は不要です。

## ステップ1：インストール（所要時間：約3分）

掲示板に掲載しているインストーラーを実行します。

1. `ICCardManager_Setup_x.x.x.exe` をダブルクリック
2. 画面の指示に従って進めます
3. **部署の選択**で該当する部署種別を選びます

| 部署種別 | 対象 |
|---|---|
| 市長事務部局 | 市長事務部局に属する課 |
| 企業会計部局 | 水道局、交通局等 |

4. インストール完了後、自動的にアプリが起動します

> Windows SmartScreen の警告が表示された場合は「詳細情報」→「実行」をクリックしてください。

---

## ステップ2：職員の登録（F2）

交通系ICカードを使う職員を登録します。

```
「職員管理」ボタン（F2）をクリック → 「新規登録」
```

1. **氏名**及び**職員番号**を入力します（備考は任意）
2. **職員証をICカードリーダーにタッチ**します
   - カード番号が自動で読み取られます
3. 「保存」をクリック

**これを職員の人数分くり返します。**

> 全員分をまとめて登録する必要はありません。後から追加登録できます。

---

## ステップ3：交通系ICカードの登録（F3）

管理対象の交通系ICカードを登録します。

```
「交通系ICカード管理」ボタン（F3）をクリック → 「新規登録」
```

### 3-1. カード情報の入力

1. **交通系ICカードをリーダーにタッチ**します
   - カード番号と残高が自動で読み取られます
2. **カード種別**を選択します（はやかけん / nimoca / SUGOCA 等）
3. **管理番号**を入力します（例：001）
   - 空欄の場合は自動採番されます
4. 「保存」をクリック

### 3-2. 登録方法の選択

「カード登録方法の選択」ダイアログが表示されます。

**→「紙の出納簿からの繰越」を選択してください。**

| 設定項目 | 入力する値 |
|---|---|
| **繰越月** | **2月** |
| **開始ページ** | **1** |
| 繰越額 | そのまま（カードの現在残高が自動入力されています） |

> **3月中に登録を行うので、紙の出納簿で管理が完了している直近の月＝「2月」を繰越月に指定します。** 開始ページは新年度の最初なので「1」です。
>
> 繰越額はカードの現在残高がデフォルトで入力されています。3月に入ってカードを利用済みの場合、実際の2月末時点の残高とは異なりますが、3月はおためし期間ですのでそのままで構いません。

「OK」をクリックすると登録完了です。

**これをカードの枚数分くり返します。**

---

## ステップ4：試しに使ってみる

登録が終わったら、実際に貸出・返却を試してみましょう。

<table style="width: 100%; border-collapse: collapse;">
<tr>
<td style="width: 50%; vertical-align: top; padding: 15px; border-right: 2px solid #ccc;">

### 貸出（カードを借りる）

1. **職員証**をタッチ
2. **交通系ICカード**をタッチ

画面がオレンジ色になり「ピッ」と鳴れば成功です。

</td>
<td style="width: 50%; vertical-align: top; padding: 15px;">

### 返却（カードを返す）

1. **職員証**をタッチ
2. **交通系ICカード**をタッチ

画面が水色になり「ピピッ」と鳴れば成功です。

</td>
</tr>
</table>

> **間違えても大丈夫！** 30秒以内にもう一度同じカードをタッチすると、操作を取り消せます。

---

## 4月1日からの運用

3月のデータは自動的に「前年度繰越」として処理されます。**特別な操作は必要ありません。**

4月以降は、普段どおり職員証→交通系ICカードの順にタッチするだけで、利用履歴が自動で記録されていきます。

物品出納簿を出力するには、「帳票」ボタン（F1）から対象月とカードを選んで「作成」をクリックします。

---

## 困ったときは

| 症状 | 対処法 |
|---|---|
| カードをタッチしても反応しない | カードをリーダーの中央に置き、数秒間動かさずに保持してください |
| 「未登録のカードです」と表示される | そのカードがまだ登録されていません。ステップ2または3の手順で登録してください |
| 画面の文字が小さい | 「設定」（F5）→「文字サイズ」で大きくできます |
| 操作方法がわからない | アプリ内の「ヘルプ」（F7）からマニュアルを参照できます |

その他のお困りごとは、お気軽にお問い合わせください。
