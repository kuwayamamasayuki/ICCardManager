# 共有フォルダモード 実機テスト手順書

**対象バージョン**: v2.1.2 以降
**テスト環境**: PC 6台（各台にPaSoRi接続）、職員証6枚、交通系ICカード6枚

---

## 事前準備

### 準備1: 共有フォルダの作成（PC-1で実施）

1. PC-1の任意の場所にフォルダを作成する（例: `D:\share\ICCardManager`）
2. フォルダを右クリック →「プロパティ」→「共有」タブ →「共有...」
3. 「Everyone」を追加し、アクセス許可を「読み取り/書き込み」に設定
4. 「共有」をクリック
5. 表示されるネットワークパスを控える（例: `\\PC-1\share\ICCardManager`）

### 準備2: 共有フォルダへのアクセス確認（PC-2～PC-6で実施）

1. エクスプローラーのアドレスバーに `\\PC-1のコンピュータ名\share\ICCardManager` を入力
2. フォルダが開けることを確認
3. 開けない場合はネットワーク設定を見直す

### 準備3: アプリのインストール（PC-1から順に全台で実施）

#### PC-1（1台目）

1. `ICCardManager_Setup_2.1.2.exe` を実行
2. 部署の選択: 該当する部署を選択
3. **データベースの保存先**: 「共有フォルダで複数PCから使用」を選択
4. パス欄に**ローカルパス**（例: `D:\share\ICCardManager`）を入力
   - PC-1は自分のローカルドライブなのでドライブレター形式でOK
5. インストール完了 → アプリが起動する
6. メイン画面が表示されることを確認

#### PC-2～PC-6（2台目以降）

1. `ICCardManager_Setup_2.1.2.exe` を実行
2. 部署の選択: **PC-1と同じ部署**を選択
3. **データベースの保存先**: 「共有フォルダで複数PCから使用」を選択
4. パス欄に**UNCパスまたはマップ済みドライブ**（例: `\\PC-1\share\ICCardManager`）を入力
   - 「参照...」ボタンでネットワーク上のフォルダを選択してもOK
5. インストール完了 → アプリが起動する
6. メイン画面が表示されることを確認
7. ステータスバーに「🔗 共有モード」が表示されていることを確認

> **チェックポイント**: 6台全てが起動し、エラーなくメイン画面が表示されればOK

## テスト用データの登録（PC-1で実施）

### 職員の登録

1. PC-1で「職員管理」ボタン（F2）をクリック
2. 「新規登録」をクリック
3. 氏名: `テスト職員A`、職員番号: `T001`
4. 職員証1枚目をリーダーにタッチ → カード番号が読み取られる
5. 「保存」をクリック
6. 同様に残り5名分を登録（`テスト職員B`～`テスト職員F`、`T002`～`T006`、職員証2～6枚目）

### 交通系ICカードの登録

1. PC-1で「交通系ICカード管理」ボタン（F3）をクリック
2. 「新規登録」をクリック
3. 交通系ICカード1枚目をリーダーにタッチ → カード番号と残高が読み取られる
4. カード種別を選択、管理番号: `001`
5. 「保存」→「新規登録として登録」を選択 → 「OK」
6. 同様に残り5枚分を登録（管理番号 `002`～`006`）

> **チェックポイント**: PC-1に職員6名・カード6枚が登録された状態

## テストシナリオ

### テスト1: データ反映の確認

**目的**: PC-1で登録したデータが他のPCに反映されるか

**手順**:

1. PC-2～PC-6のメイン画面を確認する
2. 30秒ほど待つ（定期リフレッシュのタイミング）

**期待結果**:

| 確認項目 | 期待値 |
|---------|--------|
| PC-2～PC-6のダッシュボードにカード6枚が表示される | ○ |
| PC-2で「職員管理」（F2）を開くと職員6名が表示される | ○ |
| PC-3で「交通系ICカード管理」（F3）を開くとカード6枚が表示される | ○ |

**結果**: ○ / ×（          ）

### テスト2: 基本の貸出・返却（1台のPC）

**目的**: 共有DB上での基本的な貸出・返却が正常に動作するか

**手順**:

1. PC-2で、職員証Aをタッチ → 「交通系ICカードをタッチしてください」と表示される
2. 交通系ICカード1をタッチ → 画面がオレンジ色になり「ピッ」と鳴る（貸出成功）
3. ダッシュボードの貸出中カード一覧にカード1が表示される
4. PC-2で、職員証Aをタッチ → 交通系ICカード1をタッチ → 画面が水色になり「ピピッ」と鳴る（返却成功）

**期待結果**:

| 確認項目 | 期待値 |
|---------|--------|
| 貸出時にオレンジ画面+「ピッ」 | ○ |
| 返却時に水色画面+「ピピッ」 | ○ |
| 貸出後、ダッシュボードに貸出中カードが表示される | ○ |
| 返却後、貸出中カードが消える | ○ |

**結果**: ○ / ×（          ）

### テスト3: クロスPC貸出・返却

**目的**: PC-Aで貸出したカードをPC-Bで返却できるか（最重要ユースケース）

**手順**:

1. **PC-2で貸出**: 職員証Aをタッチ → 交通系ICカード1をタッチ → 貸出成功
2. 30秒待つ（PC-3への反映を待つ）
3. **PC-3で返却**: 職員証Bをタッチ → 交通系ICカード1をタッチ → 返却成功
4. PC-2のダッシュボードを確認（30秒待つ）

**期待結果**:

| 確認項目 | 期待値 |
|---------|--------|
| PC-2で貸出成功（オレンジ画面） | ○ |
| PC-3に貸出中カードとして反映される（30秒以内） | ○ |
| PC-3で返却成功（水色画面） | ○ |
| PC-2のダッシュボードから貸出中カードが消える（30秒以内） | ○ |
| PC-3で履歴を表示すると利用履歴が記録されている | ○ |

**結果**: ○ / ×（          ）

### テスト4: 複数PCでの同時貸出

**目的**: 複数PCで同時にカードタッチした場合にエラーにならないか

**手順**:

1. **PC-2・PC-3・PC-4で同時に操作する**（3人で合図を合わせる）
   - PC-2: 職員証Aをタッチ → 交通系ICカード1をタッチ
   - PC-3: 職員証Bをタッチ → 交通系ICカード2をタッチ
   - PC-4: 職員証Cをタッチ → 交通系ICカード3をタッチ
2. 3台全てで貸出結果を確認

**期待結果**:

| 確認項目 | 期待値 |
|---------|--------|
| PC-2で貸出成功 | ○ |
| PC-3で貸出成功 | ○ |
| PC-4で貸出成功 | ○ |
| エラーメッセージが表示されない | ○ |
| 30秒後、全PCのダッシュボードに3枚の貸出中カードが表示される | ○ |

**結果**: ○ / ×（          ）

> **補足**: 一部のPCで数秒の遅延が発生する場合がありますが、最終的に全て成功すれば正常です。

| PC-3でカード2の返却成功（水色画面） | ○ |
| PC-5でカード4の貸出成功（オレンジ画面） | ○ |
| PC-6でカード5の貸出成功（オレンジ画面） | ○ |
| エラーメッセージが表示されない | ○ |
| 30秒後、全PCのダッシュボードで貸出中カード = カード3・4・5 | ○ |

**結果**: ○ / ×（          ）

### テスト8: ネットワーク切断と復帰

**目的**: ネットワークが切断された場合の動作と復帰を確認

**手順**:

1. PC-6のLANケーブルを抜く（またはWi-Fiを切断する）
2. 30秒～1分待つ
3. PC-6のステータスバーまたは警告エリアを確認
4. LANケーブルを接続し直す（またはWi-Fiを再接続する）
5. さらに30秒～1分待つ
6. PC-6でカードタッチ操作を試みる

**期待結果**:

| 確認項目 | 期待値 |
|---------|--------|
| ネットワーク切断後、「接続が切断されています」の警告が表示される | ○ |
| ネットワーク復帰後、警告が消える | ○ |
| 復帰後にカードタッチ操作が正常に行える | ○ |

**結果**: ○ / ×（          ）
