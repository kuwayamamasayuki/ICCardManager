## 3. 画面詳細設計

### 3.1 メイン画面（MainWindow）

#### 3.1.1 画面レイアウト

```
+----------------------------------------------------------------------------+
| [ヘッダー]                                                                  |
| 交通系ICカード管理システム：ピッすい    2024/04/01 09:30  [設定][帳票][管理] |
+----------------------------------------------------------------------------+
|                                                     |  [サイドバー]         |
|                                                     |  貸出中のカード       |
|                    [メインエリア]                    |  ・はやかけん H001    |
|                                                     |    4/1 9:00～         |
|                       💳                            |  ・nimoca N002        |
|             職員証をタッチしてください                |    4/1 8:30～         |
|                                                     |                       |
|                    残り 58 秒                        |  [警告]               |
|                                                     |  ・H003 残高不足      |
+----------------------------------------------------------------------------+
| [操作ヒント] 職員証→ICカードで貸出/返却   | [Esc]キャンセル               |
+----------------------------------------------------------------------------+
| 状態: 職員証タッチ待ち | 🔗 共有モード | 貸出中: 2枚                         |
+----------------------------------------------------------------------------+
```

> **共有モードインジケーター**: 共有フォルダモード（`database_config.txt` にパスが設定されている場合）で動作中の場合、ステータスバーに「🔗 共有モード」を表示する。スタンドアロンモード時は非表示。

> **ネットワーク切断警告バナー**: 共有フォルダモードでDB接続が失われた場合、メインエリア上部に警告バナー（`WarningType.DatabaseConnectionLost`）を表示する。背景色: #FFEBEE（薄い赤）。接続が回復すると自動的に非表示となる。

#### 3.1.2 領域説明

| 領域 | 説明 |
|------|------|
| ヘッダー | システム名、現在日時、各種ボタン |
| メインエリア | 状態アイコン、ステータスメッセージ、タイムアウト表示 |
| サイドバー | 貸出中カード一覧、警告メッセージ |
| カードダッシュボード | 登録カードの残高・貸出状態・最終利用日を一覧表示。並び替え機能あり |
| 操作ヒント | 操作方法のガイダンス |
| ステータスバー | 現在の状態、貸出中カード数、共有モードインジケーター |

#### 3.1.3 状態別表示

| 状態 | アイコン | メッセージ | 背景色 |
|------|----------|------------|--------|
| 職員証タッチ待ち | 💳 | 職員証をタッチしてください | 白 |
| 交通系ICカードタッチ待ち | 🚃 | 交通系ICカードをタッチしてください | 白 |
| 貸出完了 | 🚃→ | 貸出しました | #FFE0B2（薄いオレンジ） |
| 返却完了 | 🏠← | 返却しました | #B3E5FC（薄い水色） |
| エラー | ⚠️ | エラーメッセージ | #FFEBEE（薄い赤） |

#### 3.1.4 カードダッシュボードの並び替え

カードダッシュボードは以下の基準で並び替え可能。

| 並び替え | 説明 |
|----------|------|
| カード名順 | カード種別・管理番号順（デフォルト） |
| 残高（少ない順） | 残高が少ないカードを先頭に表示 |
| 残高（多い順） | 残高が多いカードを先頭に表示 |
| 最終利用日順 | 最後に使用されたカードを先頭に表示 |

### 3.2 状態遷移図

```mermaid
stateDiagram-v2
    [*] --> WaitingForStaffCard: 起動

    WaitingForStaffCard --> WaitingForIcCard: 職員証タッチ
    WaitingForStaffCard --> 履歴表示: ICカードのみタッチ

    WaitingForIcCard --> Processing: ICカードタッチ
    WaitingForIcCard --> WaitingForStaffCard: タイムアウト(60秒)
    WaitingForIcCard --> WaitingForStaffCard: Escキー

    Processing --> LendComplete: 貸出処理成功
    Processing --> ReturnComplete: 返却処理成功
    Processing --> ErrorState: エラー

    LendComplete --> WaitingForStaffCard: 3秒後
    ReturnComplete --> BusStopInput: バス利用あり
    ReturnComplete --> WaitingForStaffCard: バス利用なし

    BusStopInput --> WaitingForStaffCard: 入力完了

    ErrorState --> WaitingForStaffCard: 3秒後
```

#### 状態説明

| 状態 | 説明 | タイムアウト |
|------|------|--------------|
| WaitingForStaffCard | 職員証タッチ待ち | なし |
| WaitingForIcCard | ICカードタッチ待ち | 60秒 |
| Processing | 処理中 | なし |
| LendComplete | 貸出完了 | 3秒 |
| ReturnComplete | 返却完了 | 3秒 |
| ErrorState | エラー表示 | 3秒 |
| BusStopInput | バス停入力待ち | なし |

```
| [保存]  [キャンセル]                             |

```

#### 設定項目

設定項目は `Models/AppSettings.cs` のプロパティ順に対応する。

| 項目 | プロパティ | 設定値 | デフォルト |
|------|-----------|--------|------------|
| 残高警告閾値 | `WarningBalance` | 0～20,000円 | 10,000円 |
| バックアップ先フォルダ | `BackupPath` | 任意のフォルダパス | アプリケーションフォルダ |
| 文字サイズ | `FontSize` | 小/中/大/特大 | 中 |
| 最終VACUUM実行日 | `LastVacuumDate` | 自動記録（UI非表示） | NULL |
| メインウィンドウ位置・サイズ | `MainWindowSettings` | 自動記録（UI非表示） | 既定 |
| 音声モード | `SoundMode` | 効果音/音声（男性）/音声（女性）/無し | 効果音 |
| トースト通知の表示位置 | `ToastPosition` | 右上/左上/右下/左下 | 右上 |
| 部署種別 | `DepartmentType` | 市長事務部局（役務費）/企業会計部局（旅費） | 市長事務部局 |
| 返却時のバス停入力スキップ | `SkipBusStopInputOnReturn` | 有効/無効 | 無効 |
| 帳票出力先フォルダ | `ReportOutputFolder` | 任意のフォルダパス | 空（毎回ダイアログで選択） |
| データベースパス | （`database_config.txt`） | フォルダパス（UNCパス対応） | 空（ローカルデフォルト） |

> `LastVacuumDate` および `MainWindowSettings` はアプリが内部的に保存する状態であり、設定ダイアログには表示しない。
> データベースパスは settings テーブルではなく `C:\ProgramData\ICCardManager\database_config.txt` で管理する。

> **データベースパス設定**: データベースファイルの保存先フォルダを指定する。UNCパス（例: `\\server\share\ICCardManager`）を指定すると共有フォルダモードとなり、複数PCからの同時利用が可能となる。空の場合はローカルの`%PROGRAMDATA%\ICCardManager`が使用される。フォルダ選択ダイアログで指定可能。設定変更後はアプリケーションの再起動が必要。

### 3.7 履歴表示エリア（メインウィンドウ内）

メインウィンドウ内に埋め込まれた利用履歴の表示エリア。交通系ICカードをタッチすると表示される。

```
+--------------------------------------------------+
| はやかけん 5042  現在残高: 76円   [✕ 閉じる]     |
+--------------------------------------------------+
| 表示期間: [2026年3月]  [先月] [今月] [その他の月…]|
|            [＋ 行を追加] [📦 統合] [↩ 元に戻す]  |
| +----------------------------------------------+ |
| |☐| 日付  | 摘要              | 受入 | 払出   | |
| |--|-------|-------------------|------|--------| |
| |☐| 4/1   | 鉄道（博多～天神）|      | 260    | |
| |☐| 4/1   | バス（★）        |      | 200    | |

| +----------------------------------------------+ |
| ⏮ ◀ ページ 1/1 ▶ ⏭    表示件数: [50] 全3件   |
+--------------------------------------------------+
```

#### 機能

- 月単位の期間選択（先月・今月のクイック選択、その他の月の任意選択）

- ページネーション（表示件数選択付き）

- 行の手動追加・変更・詳細表示

- 「タッチ実行」で仮想タッチを実行

#### 項目定義

| 項目 | 説明 |
|------|------|
| 交通系ICカード | シミュレーション対象のカードをComboBoxで選択 |
| 職員 | タッチする職員をComboBoxで選択 |
| 現在残高 | カードの現在残高を数値入力 |

#### 履歴エントリの列定義

| 列 | 編集可否 | 説明 |
|----|----------|------|
| 日付 | ○ | 利用日（DatePicker） |
| 乗車駅 | ○ | 乗車駅名（テキスト入力） |
| 降車駅 | ○ | 降車駅名（テキスト入力） |
| 金額 | ○ | 利用額またはチャージ額 |
| チャージ | ○ | チャージ行かどうか（CheckBox） |

#### 操作仕様

| 操作 | 動作 |
|------|------|
| 追加 | 履歴エントリを1件追加（最大20件） |
| 削除 | 選択した履歴エントリを削除 |
| タッチ実行 | 指定した情報で仮想ICカードタッチをシミュレーション実行 |
| 閉じる | ダイアログを閉じる |

## 4. UI/UXガイドライン

### 4.1 色の使用方針

| 用途 | 色コード | 色名 | 説明 |
|------|----------|------|------|
| 貸出完了 | #FFE0B2 | 薄いオレンジ | 暖色系で「外に出る」イメージ |
| 返却完了 | #B3E5FC | 薄い水色 | 寒色系で「戻る」イメージ |
| エラー | #FFEBEE | 薄い赤 | 警告・注意 |
| ヘッダー | #2196F3 | 青 | システムカラー |
| 操作ヒント | #E3F2FD | 薄い青 | 情報表示 |

### 4.2 アクセシビリティ対応

#### 4.2.1 色覚多様性対応
- 暖色（貸出）と寒色（返却）で色相差を明確に
- 色だけでなくアイコン・テキストでも状態を表現

#### 4.2.2 多重表現
状態は以下の4要素で伝達：
1. **色**: 背景色で視覚的に区別
2. **アイコン**: 絵文字で直感的に理解
3. **テキスト**: メッセージで明確に説明
4. **音**: フィードバック音で聴覚的に通知

#### 4.2.3 文字サイズ変更
- 小/中/大/特大の4段階
- 高齢者でも読みやすい大きさに変更可能

### 4.3 音声フィードバック

| イベント | 音 | ファイル |
|----------|-----|----------|
| 貸出成功 | ピッ | lend.wav |
| 返却成功 | ピピッ | return.wav |
| エラー | ピー | error.wav |
| 警告 | ピポ | warning.wav |

### 4.4 操作フィードバック

| 操作 | フィードバック |
|------|----------------|
| カードタッチ | 背景色変化 + 音 + メッセージ |
| ボタンクリック | ボタン無効化（二重クリック防止） |
| 処理中 | オーバーレイ + プログレスバー |
| タイムアウト | カウントダウン表示 |
