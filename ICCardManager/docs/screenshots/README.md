# スクリーンショット

このディレクトリには、README.md で使用するスクリーンショットを配置してください。

## 必要なスクリーンショット

| ファイル名 | 内容 | サイズ目安 |
|------------|------|-----------|
| `main.png` | メイン画面（待機状態） | 800x600 |
| `lend.png` | 貸出完了画面 | 600x400 |
| `return.png` | 返却完了画面 | 600x400 |
| `history.png` | 履歴照会画面 | 800x600 |
| `card.png` | カード管理画面 | 600x500 |
| `staff.png` | 職員管理画面 | 600x500 |

## スクリーンショットの撮影方法

### Windows標準機能

1. `Win + Shift + S` でスクリーンショットツールを起動
2. 対象の画面を範囲選択
3. ペイントなどで保存

### 推奨設定

- フォーマット: PNG
- 解像度: 100% スケール
- 文字サイズ: 「中（標準）」設定で撮影

## 注意事項

- 個人情報（実際の職員名やIDm）が映らないようにしてください
- テストデータを使用して撮影することを推奨します
